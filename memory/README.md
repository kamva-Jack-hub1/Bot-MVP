## 📋 **MEMORY STANDARDS**

### **File Naming Convention:**
```
YYYY-MM-DD_Type_Description_Version.extension
Example: 2026-04-01_backup_database_daily_v1.json
```

### **Backup Standards:**
1. **Frequency**: Daily for active work, weekly for archives
2. **Retention**: 30 days daily, 1 year weekly, permanent monthly
3. **Verification**: Regular restore testing
4. **Security**: Encrypted backups for sensitive data
5. **Location**: Multiple locations (GitHub + local + cloud)

### **Archive Standards:**
1. **Organization**: Chronological and categorical
2. **Metadata**: Date, version, description, owner
3. **Access**: Read-only for historical archives
4. **Preservation**: Format-agnostic storage
